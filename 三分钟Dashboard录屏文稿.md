# FP-FraudSim 三分钟 Dashboard 录屏文稿

## 1. 录制准备

建议录制分辨率为 `1920×1080`，浏览器缩放调整至 `90%`。

录制前执行一键启动：

```powershell
.\scripts\start_all.ps1 -ReplayLimit 1000 -ReplayRate 200
```

打开以下页面：

```text
Dashboard：http://localhost:8000/dashboard
Flink UI：http://localhost:8081
```

录制前确认：

1. 实时页可以拉取到风险消息。
2. 图挖掘页存在 Top 风险团伙。
3. 用户画像、模型、指标和试算页面可以正常打开。
4. 录制时不执行耗时较长的模型重训或重新挖掘，只展示对应功能入口。

## 3. 录屏操作重点

### 实时页

选择一条理由码较多的交易，并展示链路图节点拖动、滚轮缩放和空白区域平移。

如果实时页没有显示交易，依次点击：

```text
回放交易流 -> 等待任务成功 -> 拉取
```

### 图挖掘页

选择风险较高且节点数量适中的团伙，展示团伙成员与共享设备、IP 等证据关系。

点击用户节点后，停留约两秒展示用户画像抽屉，再进入完整用户画像页。

### 用户画像页

展示综合多样性排序、不同来源用户、风险雷达图和图团伙关系。可以输入页号并跳转，说明系统支持浏览大规模用户画像。

### 模型与指标页

模型页展示多模型效果和热切换入口。指标页调整阈值后点击“评估阈值”，展示拦截量、误报、召回和审核工作量变化。

### 试算页

点击“填入样例”和“计算风险”，结尾停留在风险等级、决策和理由码区域约两秒。

## 4. 录制注意事项

1. 页面切换动作保持平稳，每次点击后停留一至两秒。
2. 鼠标移动应跟随讲解内容，避免快速晃动。
3. 不在录制过程中执行模型重训、GraphSAGE 训练或重新图挖掘。
4. 如果三分钟时间不足，可缩短用户画像和模型页停留时间，但优先保留实时检测和图挖掘演示。
5. 最终成片建议控制在 `2分50秒–3分10秒`。
