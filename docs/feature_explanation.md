# 多维特征来源与作用

| 特征维度 | 来源 | 构建方式 | 主要作用 |
|---|---|---|---|
| 交易基础特征 | 交易事件 | 金额、类型、渠道、支付方式、跨境标记 | 描述单笔交易形态 |
| 时间窗口特征 | Kafka/Flink 事件流 | 用户 5 分钟、1 小时交易次数与金额等 | 识别短时高频和突发行为 |
| 用户画像 | 历史交易画像 | 平均金额、账户年龄、常用设备/IP | 判断当前交易是否偏离历史习惯 |
| 设备/IP 指纹 | 设备、IP 画像 | 绑定用户数、代理/VPN、模拟器标记 | 识别多账号、设备农场和代理网络 |
| 商户画像 | 商户历史交易 | 用户数、投诉率、拒付率、金额集中度 | 识别商户套现和集中攻击 |
| 基础图统计 | 交易异构图 | 度数、欺诈邻居数量、欺诈边比例 | 描述实体局部风险邻域 |
| 团伙挖掘特征 | 图分析模块 | 共享设备/IP/商户/收款方、团伙风险分 | 识别协作式欺诈并提供证据链 |

主模型默认使用稳定的基础图统计；团伙挖掘结果作为旁路解释与高危告警，避免直接入模造成指标扰动。
